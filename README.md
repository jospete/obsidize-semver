# @obsidize/semver

Middleware to compare semver strings and not cause error explosions on malformed input.

## Installation

- npm:

```bash
npm install --save @obsidize/semver
```

- git:

```bash
npm install --save git+https://github.com/jospete/obsidize-semver.git
```
